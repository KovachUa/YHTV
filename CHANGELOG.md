# Історія розробки (Changelog) - YHTV

Цей файл містить історію змін та опис того, що було зроблено під час розробки проєкту YHTV.

## [Сесія 1] - Візуальний прототип: Сторінка "Канали"
**Дата:** Липень 2026

### Що було зроблено:
1. **Ініціалізація проєкту та концепт:**
   - Створено файл `youtube_downloader_ideas.md` з детальним описом мети проєкту, обраного стеку технологій (Rust + Vanilla JS) та структури БД.
   - Визначено, що перший етап — це виключно візуальний прототип (UI/UX), без бекенду.

2. **Створення базової структури фронтенду:**
   - Створено директорію `frontend/`.
   - Додано базові файли: `index.html`, `style.css`, `app.js`.

3. **Верстка сторінки "Канали" (`index.html`):**
   - **Навігаційна панель (Header):** Додано логотип YHTV, основні вкладки (All Videos, Channels, Queue, Config) та кнопки швидких дій (Пошук, Налаштування, Додати відео).
   - **Головний контент:** Створено заголовок сторінки з перемикачем "Show subscribed only" та кнопками для зміни вигляду (сітка/список).
   - **Картки каналів:** Зверстано сітку карток каналів (mock-дані). Картка містить банер, аватарку, назву каналу, кількість підписників, дату останнього оновлення та кнопку підписки/відписки.

4. **Інтерактивні елементи та скрипти (`app.js`):**
   - Зверстано **модальне вікно** "Add New Video or Channel" з полем для вводу URL.
   - Додано базову логіку на JavaScript для відкриття та закриття цього модального вікна при кліку на відповідні кнопки.

5. **Стилізація (`style.css`):**
   - Написано CSS-стилі (Vanilla CSS), створено темну тему, що відповідає концепту.
   - Налаштовано Flexbox/Grid для правильного та чуйного відображення карток та навігації.

6. **Доопрацювання логіки інтерфейсу (Оновлення):**
   - Додано обробники подій у `app.js` для перемикання режимів відображення: **Сітка (Grid)** та **Список (List)**.
   - Додано логіку для перемикача **"Show subscribed only"**, який приховує/показує канали за допомогою data-атрибутів (`data-subscribed`).
   - Написано CSS-стилі для режиму списку (`.list-view`), забезпечуючи компактне горизонтальне відображення карток.

7. **Внутрішня сторінка каналу та відео-картки:**
   - Додано `single-channel-view` (відображення вибраного каналу з великим банером та його відеороликами).
   - Зверстано новий дизайн карток для відео (`.video-card`) з прев'ю, тривалістю та метаданими.
   - Реалізовано перехід між головним списком каналів та конкретним каналом без перезавантаження сторінки.
   - Додано логіку для кнопок Subscribe/Unsubscribe.

8. **Логіка видалення контенту:**
   - Динамічно додано іконку видалення (кошик) на всі картки каналів та відео (з'являється при наведенні).
   - Створено глобальне модальне вікно попередження (`#delete-modal`) перед остаточним видаленням.
   - Реалізовано JS-логіку зникання картки (анімація opacity+scale) після підтвердження видалення.

9. **Вкладка Download Queue (Черга):**
   - Зверстано сторінку черги зі статусами `downloading`, `queued`, `completed`, `error`.
   - Впроваджено **Optimistic UI** (миттєва візуальна реакція).
   - Додано кнопки: "Pause All", "Clear Completed", "Clear All", "Retry" та індивідуальні іконки скасування.
   - Усі глобальні видалення підв'язані до модального вікна безпеки.

10. **Головна сторінка та Навігація:**
    - Зверстано вкладку **All Videos** і зроблено її головною сторінкою за замовчуванням.
    - Додано підтримку **localStorage**, щоб після оновлення сторінки (F5) зберігалася остання відкрита вкладка.
